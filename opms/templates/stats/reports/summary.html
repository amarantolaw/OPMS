About Apple Statistics

Apple provides a weekly report covering a four-week period and contains multiple worksheets, each including information about a different week of site usage. The main worksheet, Summary, contains a week-by-week overview of our site activity.

iTunes U Public Site Manager captures weekly report data using a Sunday through Saturday schedule, Pacific Time. Sunday is the start of the report week. iTunes U Public Site Manager displays Sundayâ€™s date in the individual worksheet tabs and the Summary worksheet columns. For example, if our site was created on Monday, 2010-09-20, our first weekly report would use the start date of Sunday and display 2010-09-19 in the individual worksheet tabs and Summary worksheet column. Apple emails our detailed Excel spreadsheet report each Wednesday evening (GMT).

More information can be found at http://deimos.apple.com/rsrc/doc/UsingiTunesUPublicSiteManager/UnderstandingiTunesUreports/chapter_13_section_2.html#//apple_ref/doc/uid/iTUPSM-CH16-SW3.
